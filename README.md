Este repositório contem exercícios feitos no Google Colab com Python, usando as ferramentas de Análise de dados.

Os exercícios foram montados pelo professor e dentro de cada um deles fiz uma tarefa diferente. No Atividade, realizei etapas de análise dados como: Extração dos dados, Verificação inicial dos dados,
limpeza e tratamento, levantamento de KPI´s e insights de negócio através de filtros e agrupamentos.
Ao final, desenvolvi dashboards com o resultado da análise e respondi as perguntas de negócio pedidas

No segundo desafio, também desenvolvido pelo professor, realizei as mesmas coisas para um segundo dataset, porém, este foi mais focado em resultados de negócio, aprendizado de filtragens, agrupamentos e correlações.

O terceiro desafio, eu precisei realizar vários tipos de limpeza de dados, seja de valores ausentes, nulos e duplicados. Aprendi que esta é a etapa mais importante da análise.
Os dados precisam estar limpos e condizentes com o negócio, para que a etapa de Análise Exploratória e a etapa de Visualização sejam verdadeiras, consistentes e mostrem a realidade daqueles dados.
